description: 작업 변경분 커밋 (메시지 자동 생성 + 사용자 승인)
argument-hint: (선택) 추가 메시지/지시
---

`.harness/docs/git-commit/index.md` 의 절차를 그대로 따른다. `$ARGUMENTS` 가 비어있지 않으면 사용자가 추가로 전달한 컨텍스트(메시지 보강·특정 파일 지정 등)로 해석.

이 슬래시 커맨드는 한국어 키워드 `작업내용 커밋해줘` 와 동등한 진입점이다.

⚠️ 변경분이 없으면 진입하지 않고 중단. 매 호출마다 staged 파일 목록 + 커밋 메시지를 사용자에게 확인받은 뒤에만 `git commit` 실행.
